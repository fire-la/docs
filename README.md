# FIREla Docs

Documentation site for [FIREla](https://firela.io) — AI advisor for Financial Independence, Retire Early.
Self-hosted financial data sovereignty.
Open source. Free. Zero lock-in..

Built with [Mintlify](https://mintlify.com) and deployed at **docs.firela.io**.

## What's inside

- **Guides** — quickstart, installation, Beancount basics, self-hosting
- **API Reference** — auto-generated from `api-reference/openapi.json` (OpenAPI 3.0)
- **Blog** — articles on self-hosting and plain-text accounting

## Local development

```bash
npm i -g mint
mint dev
```

Open http://localhost:3000.

## Deployment

Push to `main` — Mintlify GitHub app deploys automatically.
